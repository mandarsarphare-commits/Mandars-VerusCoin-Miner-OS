# 🚀 Mandar's VerusCoin Miner OS

A lightweight VerusCoin mining management system for Android Termux.

## Features

- Start miner
- Stop miner
- Check miner status
- Live dashboard
- Configurable settings

## Installation

```bash
git clone https://github.com/mandarsarphare-commits/Mandars-VerusCoin-Miner-OS.git
cd Mandars-VerusCoin-Miner-OS
python start.py
```

## Commands

```bash
python start.py
python stop.py
python status.py
python dashboard.py
```
